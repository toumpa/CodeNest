Φτιάχνω την πρώτη μου Αριθμομηχανή στην γλώσσα C

Γειά σου! Hρθε η ώρα να δοκιμάσεις να φτιάξεις το πρώτο σου πρόγραμμα σε C

Σε αυτή τη σελίδα, θα δούμε παρέα πώς μπορούμε να φτιάξουμε μια πολύ απλή αριθμομηχανή (calculator) χρησιμοποιώντας τη γλώσσα C. Το κουμπιουτεράκι μας θα μπορεί να κάνει κάπoιες βασικές, απλές πράξεις όπως πρόσθεση, αφαίρεση, πολλαπλασιασμό, διαίρεση, δύναμη, ρίζα, ημίτονο και συνημίτονο.

Βήμα 1: Δημιουργία αρχείου

Αρχικά, χρειάζεται να δημιουργήσουμε ένα νέο αρχείο με την κατάληξη .c, για παράδειγμα calculator.c.

Βήμα 2: Κώδικας

Ανοίγουμε το αρχείο calculator.c και αρχίζουμε να γράφουμε τον κώδικα.

Βήμα 3: Ορισμός βιβλιοθηκών και της κύριας συνάρτησης

Ξεκινάμε με τη συμπερίληψη των απαραίτητων βιβλιοθηκών και τον ορισμό της κύριας συνάρτησης main:

#include <stdio.h>
#include <math.h>

int main() {

    return 0;
}

Την βιβλιοθήκη math.h δεν την έχουμε αναφέρει. Είναι μια κοινή βιβλιοθήκη της γλώσσας C με έτοιμες συναρτήσεις για διάφορους μαθηματικούς υπολογισμούς, όπως ακριβώς λειτουργεί και η βιβλιοθήκη stdio.h με τις συναρτήσεις printf() και scanf().
Αυτό το πλήθος συναρτήσεων που περιέχει, επιτρέπουν την εκτέλεση μαθηματικών πράξεων όπως η δύναμη, η τετραγωνική ρίζα, οι τριγωνομετρικές συναρτήσεις και πολλές άλλες.

Βήμα 4: Εισαγωγή τιμών από τον χρήστη

Προσθέτουμε στον κώδικα την printf και scanf για να ζητήσουμε από τον χρήστη να εισάγει τον τελεστή και τους αριθμούς της πράξης.

    char op;
    float num1, num2, result;

    printf("Eisagwgh telesth: (+, -, *, /, ^, r, s, c).\n O telesths r simbolizei thn riza, to s to hmitono kai to c to sinhmitono.\n");
    scanf(" %c", &op);
    

    if (op == 'r') {    // αν ο τελεστής είναι η ρίζα χρειαζόμαστε μόνο έναν αριθμό
        printf("Dwste enan arithmo: ");
        scanf("%f", &num1);
    } 
    else if (op == 's' || op == 'c') {  // αν είναι ημίτονο ή συνημίτονο θέλουμε γωνία
        printf("Dwste mia gwnia se aktinia: ");
        scanf("%f", &num1);
    }
    else {  // σε οποιαδήποτε άλλη περίπτωση χρειαζόμαστε 2 αριθμούς
        printf("Dwste dio arithmous: ");
        scanf("%f %f", &num1, &num2);
    }

Βήμα 5: Υπολογισμός πράξεων

Προσθέτουμε τον κώδικα για να υπολογίσουμε το αποτέλεσμα ανάλογα με τον τελεστή που έδωσε ο χρήστης χρησιμοποιώντας μια δομή πολλαπλής επιλογής if:

    if (op == '+') {     // πρόσθεση
        result = num1 + num2;
        printf("%.2f + %.2f = %.2f\n", num1, num2, result);
    } 
    else if (op == '-') {   // αφαίρεση
        result = num1 - num2;
        printf("%.2f - %.2f = %.2f\n", num1, num2, result);
    } 
    else if (op == '*') {   // πολλαπλασιασμός
        result = num1 * num2;
        printf("%.2f * %.2f = %.2f\n", num1, num2, result);
    } 
    else if (op == '/') {   // διαίρεση

        if (num2 != 0) {    // απαγορεύεται η διαίρεση με το 0
            result = num1 / num2;
            printf("%.2f / %.2lf = %.2f\n", num1, num2, result);
        } 
        else {
            printf("Den epitrepetai diairesh me to mhden (0).\n");
        }
    } 
    else if (op == '^') {   // δύναμη
        result = pow(num1, num2);
        printf("%.2f ^ %.2f = %.2f\n", num1, num2, result);
    } 
    else if (op == 'r') {   // ρίζα

        if (num1 >= 0) {    // πρέπει το υπόριζο να είναι μεγαλύτερο ή ίσο με το 0
            result = sqrt(num1);
            printf("riza: %.2f = %.2f\n", num1, result);
        } 
        else {
            printf("H riza arnhtikou arithmou den epitrepetai.\n");
        }
    } 
    else if (op == 's') {   // ημίτονο
        result = sin(num1);
        printf("sin(%.2f) = %.2f\n", num1, result);
    } 
    else if (op == 'c') {   // συνημίτονο
        result = cos(num1);
        printf("cos(%.2f) = %.2f\n", num1, result);
    }
    else {  // οποιοσδήποτε άλλος τελεστής
        printf("Mh egkiros telesths.\n");
    }

Στον κώδικα υπάρχουν μερικές πράξεις που δεν γνωρίζουμε τις οποίες χρησιμοποιούμε από έτοιμες συναρτήσεις. Ας τις δούμε μια μια:
📍pow()  ☛  Υπολογίζει την δύναμη ενός αριθμού ως pow(base, exponent) όπου βάση είναι το 1ο όρισμα και ο εκθέτης το 2ο
📍sqrt()  ☛  Υπολογίζει την τετραγωνική ρίζα ενός αριθμού ως sqrt(number)
📍 sin()  ☛  Υπολογίζει το ημίτονο ενός αριθμού ως sin(angle) όπου angle είναι η γωνία σε ακτίνια
📍cos()  ☛  Υπολογίζει το συνημίτονο ενός αριθμού ως cos(angle) όπου angle είναι η γωνία σε ακτίνια

Βήμα 6: Μετάφραση - μεταγλώττιση και Εκτέλεση

Αποθηκεύουμε το αρχείο και το μεταγλωττίζουμε χρησιμοποιώντας τον compiler της C (gcc). Ανοίγουμε ένα τερματικό και πληκτρολογούμε την παρακάτω εντολή:

gcc .\calculator.c

Για να τρέξουμε τώρα το πρόγραμμα μας, πληκτρολογούμε την παρακάτω εντολή:

.\a.exe

Αυτή είναι μια πολύ απλή αριθμομηχανή που χρησιμοποιεί τη γλώσσα C. Μπορείτε να προσθέσετε περισσότερες λειτουργίες ή να βελτιώσετε τον κώδικα με βάση τις ανάγκες σας.
