Ας γνωρίσουμε τους Πίνακες

Ήρθες σε μια πρώτη επαφή με ένα ολοκληρωμένο πρόγραμμα σε C που χρησιμοποιεί μονοδιάστατους πίνακες.
Αυτή την φορά θα ασχοληθούμε και με τους δισδιάστατους πίνακες!

Σε αυτό το project θα χρειαστεί να αποθηκεύσεις σε πίνακα κάποια δεδομένα μαθητών σχετικά με τις βαθμολογίες τους σε τρία (3) διαφορετικά μαθήματα.
Τα μαθήματα αυτά είναι -> Έκθεση , Μαθηματικά , Πληροφορική.

Α. Ζήτησε από τον χρήστη να σου δώσει τις βαθμολογίες των μαθημάτων από 5 μαθητές. Οι βαθμοί θα κυμένονται από 0 έως 20(άριστα).

B. Σύγκρινε τις βαθμολογίες τους με τον μέσο όρο του τμήματος.
average = συνολικό άθροισμα βαθμών μαθήματος / μαθητές
Κάθε μάθημα θα έχει τον δικό του μέσο όρο, τους οποίους θα πρέπει να βρεις, να αποθηκεύσεις και να τους εμφανίσεις στην οθόνη ξεχωριστά. Υπάρχουν τρία μαθήματα, άρα θα πρέπει να βρεις τρεις μέσους όρους.
Θα εμφανίζει τον μονοδιάστατο πίνακα (τριών θέσεων) με τον μέσο όρο των βαθμολογιών ταξινομημένο σε αύξουσα ή φθίνουσα σειρά, ανάλογα με την επιλογή του χρήστη, την οποία θα πρέπει να διβάσεις από το πληκτολόγιο.

Γ. Εμφάνισε πόσοι μαθητές βρίσκονται κάτω από τον μέσο όρο του μαθήματος των Μαθηματικών. O μέσος όρος του μαθήματος Μαθηματικών πρέπει να συγκριθεί με τις βαθμολογίες των μαθητών.

Δ. Εμφάνισε πόσοι μαθητές προάγωνται στην επόμενη τάξη. Ένας μαθητής μπορεί να συνεχίσει στην επόμενη τάξη αν η συνολική του βαθμολογία όλων των μαθημάτων του έχει μέσο όρο μεγαλύτερο του 15.0 και επιπλέον θα ορίζει την κλίμακα του βαθμού σε:
"Καλά" < 16
"Πολύ Καλά" > 16 και < 18
"Άριστα" > 18
Π.χ. αν ένας μαθητής έχει συνολικό μέσο όρο μαθημάτων 17,2 τότε θα πρέπει να περάσει στην επόμενη τάξη με βαθμό 17,2 (Πολύ Καλά)!

Ε. Τέλος, θα ταξινομεί τον πίνακα με τις βαθμολογίες ανά μάθημα (ανά στήλη) και θα εμφανίζει την χαμηλότερη βαθμολογία καθενός από αυτά. Δηλαδή, τον χαμηλότερο βαθμό που πήρε ο μαθητής στο συγκεκριμένο μάθημα.



**Όσο αφορά την ταξινόμηση, αυτή μπορεί να είναι είτε αύξουσα είτε φθίνουσα.
Για να το επιτύχουμε αυτό, θα κάνουμε χρήση του πιο απλού "έτοιμου" αλγορίθμου ταξινόμησης που ονομάζεται bubble sort (ταξινόμηση φυσαλίδας).
Όταν λέμε "έτοιμο" αλγόριθμο εννοούμε ότι είναι μια τυποποιημένη διαδικασία (εντολές κώδικα) η οποία μπορεί να χρησιμοποιηθεί αυτούσια σε πολλές περιπτώσης, όμως δεν δίνεται έτοιμη μέσα από κάποια βιβλιοθήκη της γλώσσας.

Βήμα 6: Ταξινόμηση Μέσων Όρων

Ρώτα τον χρήστη τη σειρά ταξινόμησης (αύξουσα ή φθίνουσα) και ταξινόμησε τον πίνακα των μέσων όρων.

📍Παρακάτω θα δεις την ταξινόμηση φυσσαλίδας (bubble sort) που έχει έναν "τυποποιημένο" κώδικα.
💠Αποτελείται από δύο εμφωλευμένες επαναλήψεις for όπως η μία προσπελαύνει τις γραμμές και η άλλη τις στήλες.
💠ξεκινά από το 0 και συνεχίζει μέχρι το 3 - 1, δηλαδή το 2. Αυτός ο βρόχος ελέγχει κάθε στοιχείο του πίνακα εκτός από το τελευταίο.
💠Η j ξεκινά από το i + 1 και συνεχίζει μέχρι το 3. Αυτός ο βρόχος συγκρίνει το στοιχείο στη θέση i με τα στοιχεία στις θέσεις i + 1 έως 2 (τελευταία θέση του πίνακα). Ο σκοπός του είναι να βρει αν υπάρχει στοιχείο πιο μικρό (για αύξουσα σειρά) ή πιο μεγάλο (για φθίνουσα σειρά) από το στοιχείο στη θέση i.
💠Αν η τιμή ενός στοιχείου είναι μεγαλύτερη (αύξουσα) ή μικρότερη (φθίνουσα) από την επόμενη, αλλάζω θέσης τα στοιχεία ώστε αυτό με την μικρότερη τιμή να πάει στην αρχή του πίνακα.

Στην συνέχεια εμφανίζουμε τον πίνακα ταξινομημένο, δηλαδή εμφανίζεται ο μέσος όρος κάθε μαθήματος.

Βήμα 8: Εξέταση Προαγωγής Μαθητών

Εξέτασε πόσοι μαθητές προάγονται στην επόμενη τάξη με βάση τον μέσο όρο τους.
Πρέπει να βρούμε τον μέσο όρο του κάθε μαθητή.

Οι μαθητές είναι αποθηκευμένοι στις γραμμές του πίνακα άρα θα χρειαστεί να προσθέσουμε τις βαθμολογίες των μαθημάτων του κάθε μαθητή (όλη την γραμμή δηλαδή) και μετά να το διαιρέσεις με το 3, αφού τόσα είναι τα μαθήματα.

Αφού βρεθεί ο Μ.Ο. κάθε μαθητή θα ελέγξουμε την κλίμακα βαθμολογίας και αν είναι άνω του 15.0 τότε ο συγκεκριμένος μαθητής προάγεται στην επόμενη τάξη. 

float total_score, average_score;    //ptosthetw sthn arxh
int passing_grade;
//Posoi mathites pernane sthn epomenh tajh
for (i = 0; i < 5; i++) {   // gia kathe mathith

    //mhdenizw mesa sthn epanalhpsh
    kathe mathiths exei allh synolikh bathmologia
    total_score = 0;    

    for (j = 0; j < 3; j++) {   //gia kathe mathima
        total_score += scores[i][j];    //athroisma bathmwn
    }

    if (average_score > 15.0) {     //pernane

            passing_grade = 1;      // kala
        } 
        else if (average_score <= 18) {
            passing_grade = 2;      // polu kala
        } 
        else {
            passing_grade = 3;      // arista
        }

        if (passing_grade == 1) {
            printf(" Kala\n");
        }
        else if (passing_grade == 2) {
            printf(" Polu Kala\n");
        }
        else if (passing_grade == 3) {
            printf(" Arista\n");
        }
    }
    else{   //den pernane
        printf("O mathiths %d den pernaei thn tajh", i+1);
    }
